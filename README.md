# sprint5-vehicles

# Análise de Anúncios de Venda de Carros

App web para exploração de dados de anúncios de carros usados nos EUA.

## Funcionalidades
- Histograma de distribuição de preços
- Gráfico de dispersão de preço vs quilometragem
- Boxplot de preços por tipo de carro
